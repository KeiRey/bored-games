# Bored Games

Koleksi game ringan berbasis HTML Canvas untuk mengusir kebosanan.

## Game

### Snake Game
Buka `games/snake.html` langsung di browser.

**Alur:** Pemain menggerakkan ular menggunakan **Arrow Keys** atau **WASD** untuk memakan makanan di arena. Setiap makanan dimakan, skor bertambah **+10** dan kecepatan meningkat. Game over saat ular menabrak dinding atau tubuhnya sendiri.

**Fitur:**
- Skor & High Score (tersimpan di localStorage)
- Kontrol keyboard (Arrow Keys / WASD)
- Kontrol tombol di layar untuk mobile
- Kecepatan meningkat seiring makan越来越多
- Pengaturan: klik ikon gear untuk mengatur kecepatan (5 level) dan jumlah buah (1-10) sekaligus

**Teknologi:** HTML5 Canvas, JavaScript vanilla
